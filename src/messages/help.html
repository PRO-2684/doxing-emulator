== 开盒模拟器使用说明 ==
您可以通过下列任一方式使用 /dox 指令对他人进行开盒： 提供用户 ID：若 dox 指令后为一串数字，则开盒此 ID 的用户。 回复消息：若 bot 在群聊中，使用 dox 指令回复某人的消息即可对 ta 进行开盒。 转发消息：将某人的消息转发至此机器人，即可对 ta 进行开盒。 Inline 使用：在任意聊天界面提及此 bot，并提供用户 ID，则开盒此 ID 的用户。 * 如果未指定目标，则是开自己的盒。
== 错误代码 ==
ERR_INCOMPREHENSIBLE: 无法理解的消息。消息既不是转发的，也不是指令。 ERR_INVALID_ORIGIN: 转发来源不是用户，或者被隐藏。 ERR_INVOKER_IDENTIFICATION_FAILED: 无法确定指令发送者的身份。 ERR_REPLIED_IDENTIFICATION_FAILED: 无法确定被回复消息发送者的身份。 ERR_USER_IDENTIFICATION_FAILED: 无法确定指定用户的身份。可能是提供的并非 ID，或者用户不存在，或者 bot “不认识”此用户。在最后一种情况下，您可以尝试先转发此用户的消息到此 bot，从而让 bot “认识”此用户。
